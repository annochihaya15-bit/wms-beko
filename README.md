# 📦 WMS 仓储管理系统 (Frontend Web)

## 📖 项目简介
这是 WMS 仓库管理系统的前端源码，基于 **Vue 3 + Element Plus + Vite** 开发。
系统实现了现代化的悬浮卡片 UI，包含动态路由权限拦截、ECharts 数据大屏、Excel 导出等企业级功能。

> 🔗 **后端仓库地址**：[点击跳转到 GitHub 后端仓库](https://github.com/annochihaya15-bit/wms-beko)

## 🛠️ 技术栈 (Tech Stack)

| 模块 | 技术 | 说明 |
| --- | --- | --- |
| **核心框架** | Vue 3 | Composition API + Setup 语法糖 |
| **构建工具** | Vite 5.x | 秒级启动，极速热更新 |
| **UI 组件库** | Element Plus | 饿了么 Vue3 组件库 |
| **网络请求** | Axios | 二次封装，集成拦截器与全局Loading |
| **路由管理** | Vue Router 4.x | 路由守卫鉴权 |
| **数据可视化** | ECharts 5.x | 首页库存动态统计 |
